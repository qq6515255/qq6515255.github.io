### 学习结果

> 对面向对象和es6语法有了更多的了解
> 运用了和jq类似的方法去实现boll类的初始化和箭头函数
```
function Boll() {
    return new Boll.prototype.init();
}
Boll.prototype = {
    constructor: Boll,
    init:()=>{}
    ...(方法)
}
Boll.prototype.init.prototype = Boll.prototype;
```
